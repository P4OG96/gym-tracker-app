# Gym Tracker App (Android)

Eine einfache Android App zum Tracken von Workouts, Übungen und Sätzen.  
Gebaut, um strukturierter zu trainieren und gleichzeitig mehr über lokale Datenbanken und UI-Logik in Android zu lernen.

---

## Was die App kann

- Trainingstage erstellen und verwalten
- Übungen zu jedem Training hinzufügen
- Sätze mit Gewicht und Wiederholungen tracken
- Übungen als erledigt markieren
- Automatische Sortierung (erledigt vs. offen)
- Drag & Drop zum Umordnen der Übungen
- Swipe zum Löschen oder Umbenennen
- Integrierter Timer für Satzpausen
- Fortschritt bleibt lokal gespeichert

---

## Technik

- Java (Android)
- Room Database
- RecyclerView
- AlertDialog / Custom Input UI
- Handler (Timer)
- SharedPreferences

## Hinweise

- Alle Daten werden lokal gespeichert (keine Cloud)
- Fokus liegt auf einfacher und schneller Nutzung im Training
- Ist ein Lern-/Portfolio-Projekt

---

## Warum ich das gebaut habe

Ich wollte eine eigene Workout-App bauen, die genau auf meine Bedürfnisse passt, und dabei lernen, wie man mit Room Database arbeitet und komplexere UI-Interaktionen in Android umsetzt.
